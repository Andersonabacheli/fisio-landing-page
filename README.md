# Landing Page de Fisioterapia

 Projeto de Front-End criado para demonstrar habilidades em desenvolvimento web moderno.

 Visualize o projeto funcionando:  
 https://fisio-landing-page-ashy.vercel.app

##  Sobre o projeto 
Esta é uma landing page para divulgação de serviços de fisioterapia, com design **limpo, profissional e responsivo**, construída com foco em boas práticas de UI/UX e performance.

##  Tecnologias utilizadas

Principal stack:
- **React**
- **Tailwind CSS**
- **Vite**
- **Shadcn/UI**
- **Lucide Icons**
- **JavaScript**

##  O que o projeto apresenta
✔ Layout responsivo para todos os tipos de tela  
✔ Navegação clara e organizada  
✔ Estrutura visual pensada para conversão  
✔ Paleta de cores que transmite profissionalismo  
✔ Projeto otimizado para desempenho
